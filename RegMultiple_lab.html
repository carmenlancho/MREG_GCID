Queremos ajustar un modelo de regresión lineal múltiple (escribimos el modelo teórico): \[Y=\beta_0 + \beta_1 X_1 + \beta_2 X_2 + \cdots + \beta_p X_p + \varepsilon\]

Ajustamos el modelo a nuestros datos:

modelo_simple <- lm(mpg ~ wt, data = datos)
modelo <- lm(mpg ~ wt + hp + cyl, data = datos)
# modelo <- lm(mpg ~ ., data = datos) # el . indica que usamos todas las regresoras

Interpretación de los coeficientes

coef_modelo <- tidy(modelo) # más organizadito
print(coef_modelo)
# A tibble: 3 × 5
  term        estimate std.error statistic  p.value
  <chr>          <dbl>     <dbl>     <dbl>    <dbl>
1 (Intercept)    39.7      1.71      23.1  3.04e-20
2 wt             -3.19     0.757     -4.22 2.22e- 4
3 cyl            -1.51     0.415     -3.64 1.06e- 3

El intercept \(\beta_0\) vale 39.69, cuya interpretación es: el valor esperado de mpg cuando wt=0 y cyl=0.

El coeficiente \(\beta_{wt}\) vale -3.19, cuya interpretación es (recordamos que la variable wt está medida en 1000 lbs): por cada 1000 lbs que aumente wt, el target mpg disminuye 3.19 unidades, manteniendo fija la variable cyl.

El coeficiente \(\beta_{cyl}\) vale -1.51, cuya interpretación es: por cada unidad adicional de cyl, el target mpg disminuye 1.51 unidades, manteniendo fija la variable wt.
